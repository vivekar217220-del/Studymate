# Studymate

A Smart Study Assistant Web Application built on AWS Cloud.

## Project Idea
StudyMate helps students access study materials and manage
their learning journey. Students sign up with OTP email
verification, login securely, and access downloadable PDFs
for their subjects from a personal dashboard.

## Features
- OTP Login and Signup via AWS Cogito Free Service.
- Personal Student Dashboard.

## Tech Stack
- Frontend : Vite React + Tailwind CSS
- Auth     : AWS Cognito (OTP Email)
- Hosting  : Vite

## AWS Services Used
- Amazon Cognito

## Setup
1. Clone this repository
2. Run npm install
3. Add your Cognito config in src/aws/config.js
4. Run npm run dev
